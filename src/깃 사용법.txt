
# Git

 - .git 폴더가 위치한 폴더 안의 모든 내용을 관리해주는 프로그램
 - .git 폴더를 추가한 곳을 Repository라고 부른다
 - .git 폴더 내부에는 REpository의 모든 변경 내역을 기록한다 (변경 내역, 복원 지점 등)
 - .git 내부의 정보를 활용하여 해당 Repository를 예전 모습으로 되돌릴 수 있다
 - Git은 레포지토리 내부의 미세한 변화도 모두 알아채고 관리해준다
 - Eclopse에 내장된 Egit이라는 프로그램을 사용할 수 있다
 
# 새 레포지토리 만들기

 - Widow -> Show View -> Others -> Git - > Git Repository
 -> Create a new local Git repository
 - 경로 잘 정해서 잘 만들기
 
 # 새 프로젝트 생성하기
 
 - 레포지토리를 모아놓을 폴더 내부에 새프로젝트 생성하기
 
 
 
 # 레포지토리에 프로젝트 연결하기
 
  - 레포지토리 우클릭 -> Import Project
 
# Commit

 - 레포지토리 내부의 변경 사항을 확정짓는 명령어
 - 커밋한 시점은 복원 지점이 된다
 - 체크아웃 명령을 통해 언제든지 원하는 커밋 시점으로 현재 상태를 변경할 수 있다
